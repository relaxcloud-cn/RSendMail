# rsendmail-tauri

Desktop GUI for RSendMail built with Tauri, Vue 3, TypeScript, and Vite.

## Development

```bash
npm ci
npm run dev
```

## Production Build

```bash
npm ci
npm run build
cargo build --release --manifest-path src-tauri/Cargo.toml
```

## Recommended IDE Setup

- VS Code
- Vue - Official (Volar)
- Tauri VS Code extension
- rust-analyzer
